¿Cómo reportar incidencias?

  1. Debe registrarse e iniciar sesión.
  2. Una vez dentro, encontrará un formulario para reportar incidencias.

Procedimiento de Usuario — Reporte de problemas

  1. Registro en el sistema ( En caso de ser usuario nuevo debe realizar éste paso)

    • El usuario debe crear una cuenta proporcionando sus datos personales obligatorios (nombre, correo, contraseña, teléfono y cédula).
    • El sistema valida que la información sea correcta y, si cumple los requisitos, registra al ciudadano en la base de datos.

  2. Inicio de sesión

    • Una vez registrado, el usuario debe autenticarse ingresando su correo y contraseña.
    • El sistema verifica las credenciales y, si son válidas, permite el acceso al panel principal.

  3. Acceso al módulo de reportes

    • Dentro del panel, el usuario selecciona la opción:
    • Debe dirigirse al formulario “Crear reporte”
    • Esto abre el formulario de reporte ciudadano.

  4. Completar formulario de reporte

    Información obligatoria:

    • Tipo de reporte: seleccionar categoría (ej. bache, fuga de agua, luminaria dañada, alcantarilla obstruida).
    • Descripción del problema: explicar claramente la situación observada.
    • Ubicación: dirección, referencia o coordenadas.

  5. Validación de datos

    El sistema revisa que:

    • No haya campos vacíos obligatorios.
    • La descripción tenga contenido válido.
    • La ubicación esté especificada.
    • Si hay errores, se muestran mensajes indicando qué debe corregirse.

  6. Envío del reporte

    Cuando la información es válida:

    • El sistema genera un ID único de reporte.
    • Guarda los datos en la base de datos.
    • Registra fecha y hora.
    • Asocia el reporte al usuario.

  7. Confirmación al usuario

    • El sistema muestra el mensaje: “Reporte enviado correctamente”.
    • Opcionalmente también puede mostrar:
      • Número de reporte.
      • Estado inicial: Pendiente.

  8. Seguimiento posterior (flujo futuro recomendado)

    El usuario podrá:

    • Consultar sus reportes.
    • Ver estado (Pendiente, En proceso, Resuelto).
